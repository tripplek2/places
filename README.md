# Places that i've been to.

# Descrption
Below is a descrption of a form that ebables a user to input data to places they've been to and what it entails.

# Content
Create a Place object
  - Store location
  - Store landmark
  - Store time of the year
  - Store notes
  - Store image 

Create a FavouritePlaces object
  - Store all places in a list
  - Keep track of unique IDs

When the page loads
  - Create a few default places
  - Add them to FavouritePlaces
  - Display all place names on the page

When a user fills the form and clicks "Add Place"
  - Get values from the input fields
  - Check that location is not empty
  - Create a new Place object
  - Add the place to FavouritePlaces
  - Refresh the list on the page
  - Clear the input fields

When a user clicks on a place name
  - Find the place using its ID
  - Display all details for that place
  - Location
  - Landmark
  - Time of year
  - Notes
  - Image
